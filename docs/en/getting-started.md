# <span translate="no">Getting Started</span>

This section takes you through a high level overview of getting started with the **BookStop** 
booking system. Understanding the basic workflow will assist greatly when getting to grips  
configuring and using the system on a regular basis.  

## One Time Tasks

These are the initial configuration tasks that you may come back to infrequently as your business
changes:

- Get pen and paper **(!important!)** 
- Set contact information
- Set social links
- Set bookable resources
- Set booking types
- Set price bands
- Set slot templates
- Set payment policies
- Set **Stripe** (if required)

## Periodic Tasks

These are done on a regular basis to maintain slots that can be booked by customers:

- Generate provisional slots
- Publish bookable slots

## Administration Tasks

These are day-to-day tasks as bookings are serviced:

- Review bookings
- Set booking status
